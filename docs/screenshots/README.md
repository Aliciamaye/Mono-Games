### Image Specifications
- **Format**: PNG or JPG
- **Resolution**: 1920x1080 (Full HD) preferred
- **Aspect Ratio**: 16:9
- **File Size**: Under 500KB (optimized)
- **Naming**: `game-name-screenshot.png` (lowercase, hyphenated)

### GIF Requirements
- **Format**: GIF or MP4
- **Resolution**: 1280x720 or 1920x1080
- **Duration**: 5-15 seconds
- **File Size**: Under 5MB
- **FPS**: 30 or 60
- **Naming**: `gameplay-demo.gif`

## üéÆ Capturing Screenshots

### In-Game Method
1. Play the game until interesting moment
2. Press `PrtScn` (Windows) or `Cmd+Shift+4` (Mac)
3. Save to this folder
4. Rename appropriately

### Browser DevTools
1. Open game in browser
2. Press `F12` for DevTools
3. Press `Ctrl+Shift+P` and type "Screenshot"
4. Select "Capture full size screenshot"

### Recording GIFs
- **Windows**: Use OBS Studio or ScreenToGif
- **Mac**: Use QuickTime Player or Kap
- **Online**: CloudConvert for MP4 ‚Üí GIF conversion
